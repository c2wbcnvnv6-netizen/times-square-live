## Times Square Live Aggregate

Public live video dashboard for Times Square (NYC).

- Multiple simultaneous webcam views (EarthCam primary + YouTube embeds + more)
- Curated list of all major public live sources
- Current situation summary (as of June 14 2026 data: large Brazilian World Cup fan celebrations, heavy but normal police presence for event crowd control. **No riots detected**.)
- Tips for monitoring for any incidents

### Usage

1. Open `index.html` directly in a browser (best for side-by-side live views).
2. Or visit the deployed version (enable GitHub Pages or import to Vercel for a hosted URL).

### Deploy options (one-click for recipients)

**GitHub Pages (easiest free):**
- Go to repo Settings → Pages
- Source: "Deploy from a branch"
- Branch: `main`, Folder: `/ (root)`
- Save. Site appears at `https://<username>.github.io/times-square-live/`

**Vercel (recommended for previews + speed):**
- Go to https://vercel.com/new
- Import Git Repository → select this repo
- Deploy (static HTML detected automatically)
- Get a clean `*.vercel.app` URL in seconds

All camera feeds are public third-party streams. This page simply aggregates links and embeds for convenience. Always cross-check live video + official sources (NYPD, local news).
